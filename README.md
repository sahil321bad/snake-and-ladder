# 🐍 Snake & Ladder - Online Multiplayer

A real-time multiplayer Snake and Ladder game using Node.js + Socket.io.

## 🚀 How to Run

### 1. Install Dependencies
```bash
cd snake_ladder_online
npm install
```

### 2. Start the Server
```bash
npm start
```

The server will start on `http://localhost:3000`

### 3. Play with Friends

1. **Player 1**: Open browser and click "Create Room"
2. **Player 2**: Open browser, enter the room code, click "Join Room"
3. Share the room code with your friend to play together!

## 🎮 Features

- Real-time multiplayer (2 players)
- Room-based gameplay with unique codes
- Sync dice rolls and player positions
- Sound effects
- Visual snake and ladder indicators
- Win celebration
